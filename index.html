Open source CI/CD dashboard

Your CI dashboard shows what's broken.
Pung tells you what to fix.

Prescriptive CI/CD dashboard for GitHub teams. Tracks CI health, PR velocity, and merge trends — then recommends the highest-ROI action to unblock your team.

●

CI Health at a Glance

Latest run status per repo per trigger. Success rates: 7d / 30d / 90d. Avg duration trends.

▲

Merge Velocity

Track merges/day across repos. Spot downward trends before they become blockers.

◼

Stale Branch Detection

Branches with no commit in 3+ days. Paginated, sortable. Clean up dead weight.

◆

GitHub Device Flow Auth

Zero-config login. No OAuth app registration needed. Like gh auth login.

→

JSON Snapshot API

GET /api/snapshot — full dashboard state in one call. Built for agents and automation.

◯

Demo Mode

DEMO_MODE=true — try the full dashboard with realistic fake data. No GitHub needed.

Get started

Five minutes. Three lines.

Self-hosted, open source, forever free.

~
$ git clone https://github.com/arinyaho/pung.git
$ cd pung && cp .env.example .env
# set GITHUB_TOKEN, REPOS
$ docker compose up -d
 
Ready at http://localhost:2357
